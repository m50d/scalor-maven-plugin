scalor:scala-native-link-test

Full name:

com.carrotgarden.maven:scalor-maven-plugin_2.12:1.4.0.20180329171945:scala-native-link-test

Description:

Generate Scala.native runtime binary for scope=test. Provides incremental linking in M2E. A member of goal=scala-native-link.

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: test.
  • Binds by default to the lifecycle phase: process-test-classes.
