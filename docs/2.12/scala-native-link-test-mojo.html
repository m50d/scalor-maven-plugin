scalor:scala-native-link-test

Full name:

com.carrotgarden.maven:scalor-maven-plugin_2.12:1.4.0.20180304173533:scala-native-link-test

Description:

Generate Scala.native runtime binary for scope=test. A member of goal=scala-native-link. Provides incremental linking in M2E.

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: test.
  • Binds by default to the lifecycle phase: process-test-classes.
